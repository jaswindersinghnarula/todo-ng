# 📝Todo v2.0

Todo app is developed to practice my development skills. Feel free to clone this repo and make it your own.

# 👓Prerequise

Install Docker

https://docs.docker.com/engine/install/

# ⚙Get started

Edit **.env** file and run below command:

`docker compose up -d`
